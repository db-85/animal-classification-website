Animal classification¶

This code is used to train a machine learning model able to classify 151 different species of animals and more than 16,000 images. The original dataset can be found here: Animal Image Classification Dataset (kaggle.com)

Splits the data into a training (80%) and a test (20%) dataset. The splitting is stratified, so each class is represented with the same proortion in the training and validation dataset (80-20).

image_list = []
label_list = []

for subfolder in subfolders:
    file_list = os.listdir(os.path.join(dataset_folder,subfolder))
    image_list.append(['./' + subfolder + '/' + file for file in file_list])
    label_list.append([subfolder]*len(file_list))

image_list = list(itertools.chain(*image_list))
label_list = list(itertools.chain(*label_list))

x_train, x_test,y_train,y_test = train_test_split(image_list,label_list,test_size=0.2,stratify=label_list,random_state=0)


df_train = pd.DataFrame({'id' : x_train,'label' : y_train})
df_test = pd.DataFrame({'id' : x_test,'label' : y_test})

Preprocessing the training and validation datasets¶

The training and validation dataset are resized to 256x256 pixels. In addition, the training dataset is augmented by performing rotations and zooming.

Builds the CNN model and performs the fitting¶

The CNN model comprises 5 convolution/max pooling layers, 2 hidden neural layers and the output layer. The output layer will return a vector of n_class probabilities.

In [9]:
# Building the CNN model
model = models.Sequential([
    layers.Conv2D(32, (3, 3), activation='relu', input_shape=(256, 256, 3)),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(64, (3, 3), activation='relu'),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(128, (3, 3), activation='relu'),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(128, (3, 3), activation='relu'),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(128, (3, 3), activation='relu'),
    layers.MaxPooling2D((2, 2)),
    layers.Flatten(),
    layers.Dense(1024, activation='relu', kernel_regularizer=regularizers.l2(0.001)),
    layers.Dropout(0.2),
    layers.Dense(1024, activation='relu', kernel_regularizer=regularizers.l2(0.001)),
    layers.Dropout(0.2),
    layers.Dense(n_classes, activation='softmax')
])

# Compiling the model
model.compile(loss='categorical_crossentropy',
              optimizer='adam',
              metrics=['accuracy'])

# Training the model
history = model.fit(
    train_generator,
    epochs=100,
    validation_data=validation_generator
)

# Evaluating the model
test_loss, test_acc = model.evaluate(validation_generator)
print(f"Test accuracy: {test_acc}")
